# 2WD Robot Simulation

Este repositorio contiene un proyecto de Unity para simular un robot de dos ruedas (2WD). Incluye varios controladores de movimiento, modelos 3D y una escena de prueba.

## Estructura
- **Assets/**: Contiene scripts, modelos y escenas.
- **Packages/**: Gestión de paquetes de Unity.

## Controladores principales
- `CarControl`: lógica avanzada con torque y dirección adaptativa.
- `WheelsController`: ideal para control desde Python.

## Uso
1. Abrir el proyecto en Unity 2020+.
2. Ejecutar la `SampleScene`.
3. Para integración con Python, ver el script `WheelsController`.

## GitHub
Asegúrate de no subir carpetas generadas por Unity siguiendo el `.gitignore` incluido.
